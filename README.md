# Coffee Shop Website ☕

A responsive coffee shop website built using HTML, CSS, and JavaScript.

## Features

- Responsive design
- Smooth scrolling navigation
- Review form to generate instant review
- Customer feedback section
- Modern UI with animations
- Add your review through review form

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Structure

coffee-shop-website/
├── index.html
├── style.css
├── script.js
└── images/

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
